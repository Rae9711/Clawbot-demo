# Avatar Evolution Pack (One Base Lottie)

This pack is manually designed and code-friendly.

## Core rule
- One base Lottie for all levels
- Evolution is expressed by shell + effects + accessories + state overlays

## Core evolution shells
- evo-core-lv1.svg  => Lv1 云核体
- evo-core-lv5.svg  => Lv5 云核体·进阶
- evo-core-lv10.svg => Lv10 云核体·觉醒

## State overlays
- state-thinking.svg => thinking
- state-success.svg  => success
- state-error.svg    => error
- state-sleep.svg    => sleep/offline

## Usage
- Keep Lottie body as the living center.
- Place shell behind Lottie to indicate same-lifeform evolution.
- Place state overlay above Lottie for transient feedback.
